RandOp

About

RandOp is a simple website for random option selection. You can use it to decide between options, if you can't decide which is best.

Instructions

1.) Enter each option into the options field and click the add button.

2.) Choose the selection method you want using the drop down menu.

3.) Click the select button to randomly select one of the options.
